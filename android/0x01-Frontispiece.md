# Frontispiece

## About This Standard

The OWASP MASVS for Android is an Android-specific adaptation of the [OWASP Mobile Application Security Verification Standard (MASVS)](https://github.com/OWASP/owasp-masvs). It takes each platform-agnostic MASVS control and expands it into concrete, testable Android sub-requirements that reference specific APIs, manifest attributes, configuration files, and platform behaviors.

**This is NOT an official OWASP project.** It is a community fork maintained by [Jim Manico](https://github.com/jmanico) for teams that need Android-specific security requirements they can hand directly to developers, auditors, and pentesters.
